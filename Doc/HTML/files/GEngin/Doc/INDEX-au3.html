Summary

G-Engin is a simple and complet 2D game developpement UDF for AutoIt3!

It is intended for fast and headache-less game creation, because it takes care for you about all technical aspects of graphics, sounds and physics

G-Engin uses

  • For graphics: GDI+
  • Sounds: BASS
  • Physics: Custom tiny engin of mine

Objects

G-Engin is based on some basic objects (really, they are arrays)

Image

is a basic texture that can be linked with a Sprite object.

You can load several pictures format into an Image Object (bmp, gif, jpg, png, tiff, emf), you can also load only a part (Rectangle) of a picture into an Image Object.  (See _GEng_ImageLoad)

Finally, you can load a binary string representing a picture file, this is usefull to embeed your ressources in your script without the need to extract them on the hard drive.  (See _GEng_ImageLoadStream)

Sprite

(_GEng_Sprite_Create) is the main object of G-Engin, it is a advanced 2D Sprite with

  • a Texture represented by an Image Object of a part (Rectangle) of an Image Object (_GEng_Sprite_ImageSet & _GEng_Sprite_ImageSetRect)
  • Position (_GEng_Sprite_PosSet), Angle (_GEng_Sprite_AngleSet), Size (_GEng_Sprite_SizeSet)
  • Velocity (_GEng_Sprite_SpeedSet), acceleration (_GEng_Sprite_AccelSet), innertia (_GEng_Sprite_InnertieSet)
  • Angular velocity (_GEng_Sprite_AngleSpeedSet), Angular acceleration (_GEng_Sprite_AngleAccelSet), Angular innertia (_GEng_sprite_AngleInnertieSet)
  • a Shape that can be Point, Rectangle (AABB) or Circle (_GEng_Sprite_CollisionSet)
  • a Mass (used by dynamic collision calculations) (_GEng_Sprite_MasseSet)

Animation

(_GEng_Anim_Create) is a collection of animation frames (_GEng_Anim_FrameAdd), and each animation frame have

  • a Texture represented by an Image Object of a part (Rectangle) of an Image Object
  • a duration (in ms)

An Animation Object can be applied to a Sprite Object (See _GEng_Sprite_Animate) to animate it, the frames of the animation object are applied one by one to the sprite.

Font

is a font definition (Font name, size, style...) to be used by Text Object to draw text on screen (<_GEng_Font_Create>)

Text

is a rectangle containing some colored text drawed with a specified Font Object (Text Rectangle)

Sound

represents a sound file (MP3, MP2, MP1, OGG, WAV, AIFF), it hase some basic parameters (volum, pitch, pan)

Functions

In addition to these object, there are some functions

System Function

starting G-Engin systems, opening a render window, geting the FPS...  (System)

Geometry Functions

very usefull in 2D games (distances, angles, vectors...)  (Geometry functions)
