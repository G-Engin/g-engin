Tutorial 2

Introduction

In this next tutorial, we’ll learn how to use the sprite’s dynamics in order to move and rotate it automatically.

You must have read the first tutorial, ‘cause I’m not going to re-explain what have ben already done!

Coding

Well, for this tutorial, we’ll start just like the first

Global $scrW = 800, $scrH = 600

Opt("MouseCoordMode", 2)

#include <Misc.au3>
#include <..\GEngin.au3>

_GEng_Start("GEngin - Tutorial 2", $scrW, $scrH)

We then load an image, create a sprite with it, and configure the sprite

$img_Gimp = _GEng_ImageLoad("res\gengin.png", 64, 64)

$spr_Gimp = _GEng_Sprite_Create()
    _GEng_Sprite_ImageSet($spr_Gimp, $img_Gimp)
    _GEng_Sprite_OriginSetEx($spr_Gimp, $GEng_Origin_Mid)
    _GEng_Sprite_PosSet($spr_Gimp, $scrW / 2, $scrH / 2)

Some new functions now!  We assign a maximum vectorial and angular velocity

_GEng_Sprite_SpeedSet($spr_Gimp, 0, 0, 100) ; 100 pixels/s
_GEng_Sprite_AngleSpeedSet($spr_Gimp, 0, 30) ; 30 Deg/s

And vectorial and angular innertia (it’s the amount of speed that is lost per second)

_GEng_Sprite_InnertieSet($spr_Gimp, 100)
_GEng_Sprite_AngleInnertieSet($spr_Gimp, 60)
